⚙️ Compression Machine Simulator

1- Project Overview

This Python-based project simulates a compression machine, visually demonstrating the principles of material compression. It's a single-file application designed for modularity and future expansion.

3- Getting Started

To run the simulator, follow these steps on your Linux (Ubuntu) system.

    Prerequisites: Ensure you have Python 3 installed.

    Run the Simulator:

        Navigate to the project directory.

        Activate the virtual environment: source venv/bin/activate

        Run the main script: python3 main.py

        Exit: When you are finished using the simulator, you can exit the virtual environment: run deactivate on Bash.
